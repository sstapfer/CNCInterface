void loop ( )

loop function is called multiple times.

Basically it does 3 things:

  1. Processing of the webserver
  2. Processing of the current state
  3. Switching to the new state

States:

  • Init: Configuration of CNC
  • Waiting: Wait for CNC to complete the last order. Depending on previous state there are different displays on the LCD
  • Idle: Processing of the user interface
  • XY_Zero: Sets the X and Y origin 10 mm on the inner side of the PCB
  • Z_Pre_Zero: Approach the PCB as preparation for the next step
  • Z_Zero: Move drill down on the surface of the PCB
  • Startpos: Move drill to a start position 100mm above the PCB
  • Ready: Wait for user to switch on the drilling motor
  • Processing: Send the next G-Code order to the CNC machine
  • Done: Move the milling tool away and display done to the user

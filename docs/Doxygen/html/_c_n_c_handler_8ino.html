Detailed Description

These functions are used in the milling process to move the CNC and send the new orders.

Function Documentation

void initCNC ( )

this function initializes the CNC machine by

  • defining the axes XYZ
  • the speed of the different axes
  • doing a reference movement
void setXYAxesZero ( )

this function changes the X and Y origin of the coordinate system to the corner in front on the left side

void moveZAxis ( )

this function moves the drilling head 10 mm above the PCB surface and 25mm in X and Y direction

void setZAxisZero ( )

this function moves the drilling head down, until it touches the PCB. The new position is saved as Z = 0.
After resetting the Z position the head is moved away from the PCB.

void moveToStartPos ( )

this function moves away the head at about 100mm above PCB

void sendCNC ( )

this function is called to send the next order to the CNC machine. If there is no new line the order is
completed and the state changes to Done.

void moveToolAway ( )

this function moves the table in front to the user and the drilling head up.
